Sécurité & conformité

Politique d'accès des espaces institutionnels EvoSU.

Les portails Ministère, Super Admin et Admin université sont isolés du site public étudiant. L'authentification passe par l'API nationale avec cookies sécurisés et journalisation des actions sensibles.

Règles d'accès

  • Ministère — connexion uniquement via /ministere/
  • Super Admin — connexion via /superadmin/
  • Admin université — connexion via /admin-uni/ + code établissement
  • Pas d'inscription publique pour ces rôles

Bonnes pratiques

  • Changez le mot de passe par défaut après la première connexion
  • Ne partagez jamais vos identifiants institutionnels
  • Déconnectez-vous sur les postes partagés
  • Signalez toute activité suspecte au Super Admin EvoSU
